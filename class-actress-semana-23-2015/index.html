Class Actress: Semana #23 2015

Publicado por Lenny Nero el 12 de junio de 2015

El proyecto musical de Elizabeth Harper, Class Actress, encabeza el resumen de esta semana. Hace unas semanas presentaba el single ‘More Than You‘ y ahora le toca el turno a ‘High On Love‘ acompañado de videoclip. También tenemos vídeo para la canción ‘Mercy‘ de Muse. Además, Cayucas, Craft Spells o Public Access T.V.. ¡Buen fin de semana muchachada!
